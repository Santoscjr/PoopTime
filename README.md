# PoopTime
PoopTime é uma aplicação web divertida e responsiva que calcula quanto o usuário ganha enquanto está no banheiro durante o expediente. O sistema solicita salário mensal e carga horária semanal, calcula o valor por minuto e utiliza um cronômetro para mostrar em tempo real o ganho acumulado, tudo com um design leve e temático, sem ser vulgar.
